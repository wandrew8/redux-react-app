# Reduxstagram

## Overview
Reduxstagram is a React-based app with redux implementation. This application was built through the Wes Bos Redux course as a tool for continued education in React and Redux. 

## Technologies Used
* ReactJS library
* Redux
* Express
* CSS preprocessor Stylus
